# Fraud Detection Project

## Ringkasan
Project ini berfokus pada deteksi kecurangan pembayaran digital menggunakan notebook end-to-end. Notebook mencakup eksplorasi data, rekayasa fitur, penanganan data tidak seimbang, pelatihan beberapa model, evaluasi komprehensif, dan interpretasi model.

## Struktur Direktori (Saat Ini)
```
fraud-detection-project/
  .gitignore
  README.md
  01_fraud_detection_comprehensive.ipynb
```

## Cara Menjalankan
1. Buat dan aktifkan virtual environment (opsional).
2. Install dependency yang dibutuhkan di notebook.
3. Jalankan `notebooks/01_fraud_detection_comprehensive.ipynb`.
